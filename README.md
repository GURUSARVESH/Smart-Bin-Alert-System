This project is an IoT-based login authentication system built using Embedded C and the NodeMCU ESP8266 Development Board.
It provides a secure login mechanism and redirects authenticated users to a real-time dashboard for monitoring and interaction.

The system is designed for smart access control applications such as smart doors, attendance systems, and IoT dashboards.

The login page credentials will be integrated with an XAMPP-based SQL database. The SQL implementation will be added in a later update.
